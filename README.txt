Формирование скриншотов по количеству госпитализированных пациентов

Основной скрипт для запуска - main.py

Отчет в BI - hosp_covid_dis_covid_hours_for
Для выгрузки отчета необходимо иметь активное подключение vpn к домену mosreg.
Также необходимо убедиться в доступности отчета и указать свои данные логина и пароля в фале config.ini.

По умолчанию программа формирует 4 таблицы и скриншота.
    1. Количество госпитализированных за предыдущие сутки в виде большой таблицы.
    2. Количество госпитализированных за предыдущие сутки в виде сокращенной таблицы.
    2. Количество госпитализированных на текущий момент текущих суток сутки в виде большой таблицы.
    2. Количество госпитализированных на текущий момент текущих суток сутки в виде сокращенной таблицы.
При необходимости сформировать таблицу на другую дату необходимо изменить параметры в main.py при вызове класса
BI_Report_Hospital_on_hour, где date_now: date = объект дата, time_now: time = время в формате целого числа.

Для отправки сообщения в телеграмм чат нужно указать в config.ini - токен Вашего телеграмм бота и чат группы.

Важно! Каждая операционная система или ее дистрибутив по-разному генерирует изображение. В связи с чем необходимо
в файле config.ini в разделе [CUT_IMAGE] указать координаты для обрезания изображения.
